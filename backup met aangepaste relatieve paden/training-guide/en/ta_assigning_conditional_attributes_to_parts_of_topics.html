To assign conditions to a part of a topic

  1. Select the element you want to apply a conditional attribute to.
  2. In the Attributes list, click in the empty field next to a conditional processing attribute and enter a value.
  3. Press Enter.
    Tip: In Oxygen, you can see which elements you have applied an attribute to by choosing DITA > Profiling/Conditional Text > Show profiling attributes. Elements with an attribute receive a default highlight in green:
