Conditional attributes

Conditional attributes are used to filter content, flag content or perform custom processing:

Attribute Purpose
@audience Labels elements with an audience, on which rendering engines can filter. For example, "internal-use" and "customers".
@platform Labels elements with an operating system or similar environment, on which rendering engines can filter. For example, "Windows" or "Mac".
@product Labels elements with a product or component value, on which rendering engines can filter. For example, "iPhone 12" and "iPhone 12 Mini".
@rev Highlights topics with a revision level.
@props Determines new conditional processing attributes.
@otherprops Used for values that do not fit in the other default attributes.
