Handling the first column of a table as header

You can format the first column of a regular table (CALS) and simple table as a header by adding an attribute to the <table> or <simpletable> element. The attribute to mark this header column is different per type of table.
Do one of the following:
Table typeAttribute to be used
<simpletable> Set the @keycol attribute to1.
<table> Set the <rowheader> attribute to firstcol.
