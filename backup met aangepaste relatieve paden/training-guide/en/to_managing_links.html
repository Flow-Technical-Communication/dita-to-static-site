To move files inside a project

It may occur that you need to create a new (sub)folder inside a project and move existing files to that folder. When you move these files in your Explorer, you will end up with broken links. This task describes how to move files inside a project without breaking links, by making use of Master Files.
  1. Open the project and the folder you want to reorganize.
  2. In the folder you just opened, right-click the .ditamap file and choose Add to Main Files.
    The ditamap is now added to the Master files.
  3. Create the new (sub)folder.
  4. Select the files you want to move and drag them to the new (sub)folder.
  5. In the Move pane, make sure you selected the correct files and click Move.
  6. Check the box next to Update references of the moved resource(s) and click Move.
The files are now moved to the new (sub)folder and all links are still working.
