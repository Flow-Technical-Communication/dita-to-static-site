    To reference the location of the templates folder

    The location of a templates folder is to be referenced in Oxygen.

    1. Choose Options > Preferences and select for Document Type Association.
    2. Double-click DITA for Flow..
    3. Select Templates and click the plus button to add the templates directory to the list.
      Important: It is recommended to make the reference relative to the framework directory (for example, ${frameworkDir}/topic_templates)
