To refer to a figure in text

Sometimes, you may want to refer to a figure number in your text, for example "see fig. 4". All you need to do in that case, is insert a cross-reference to the figure title and set up your stylesheet to display the figure number. How you set this up in your stylesheet depends on:

  • The output format to which you publish, for example PDF or an HTML-based format such as webhelp
  • The technology you use to create your stylesheets, for example CSS, XSL-FO, MiramoPDF

In web output, you will just want to display the figure number and maybe the caption text, for example "see fig. 4". In a PDF, you may also want to include the page number, for example "see fig. 4 on page 9".

  1. In your text, insert the xref (cross-reference) element.
  2. In the Cross reference (xref) window, select the file which has the image to which you want to refer.
    You will see a list of elements which have an ID.
  3. Select the <fig> element to which you want to refer.
    Notice: Select the <fig> element, not the <image> element.
    See Figures and images.
  4. Click Insert and close.
That's all you need to do in your DITA topics. Next, you set up your stylesheets to display the text you wish to see for your cross-references to figure numbers.
