To add a DITA map to a DITA bookmap

Adding DITA maps to a bookmap is recommended as you should avoid putting <topicref> elements directly into a <bookmap>. By nesting a DITA map inside a bookmap, you create submaps which can help you manage your content.
  1. Select your bookmap.
  2. Right-click the map and select Append child > Reference.
  3. In the Insert Topic Reference dialog window, navigate to the desired DITA map and click Insert and close.
