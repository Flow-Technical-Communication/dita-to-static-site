What are the main components of a DITA map?

Component list

The following components are most commonly used in a DITA map (<map>):

Component Purpose
<title> Contains the title of your map. See Map elements.
<topicref> Links topic or another DITA map. See Map elements.
<topicgroup> Groups topics without affecting their hierarchy. See Map elements.
<topichead> Groups topics within the hierarchy. See Map elements.
<topicmeta> Contains metadata for the topic that applies specifically to the context of the publication defined by the ditamap. See Map elements.
<reltable> Specifies relationships between topics, linking them outside of the relationships defined by the hierarchy. See To create a relationship table.
