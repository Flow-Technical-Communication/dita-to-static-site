To add a DITA map to a DITA map

By nesting a DITA map inside another DITA map, you create sub-maps which can help you manage your content.
  1. Select your DITA map.
  2. Right-click the map and select Append child > Reference.
  3. In the Insert Topic Reference dialog window, navigate to the desired DITA map and click Insert and close.
  4. Save the DITA map.
