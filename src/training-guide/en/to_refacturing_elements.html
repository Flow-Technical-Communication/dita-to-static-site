Renaming elements

Parent topic: Working with DITA topics

What is renaming elements?

Renaming an element is changing that element into another element. This can be useful if you accidentally wrapped your text in incorrect tags or if you are reorganizing a topic.

For example, to promote all the paragraphs in a particular topic to sections, rename all the <p> elements to <section> elements.

To rename an element

  1. Right-click one of the element tags that surrounds the element you want to rename.
  2. Choose Refactoring > Rename Element.
  3. Enter the new name of the element or select it in the drop-down list.
  4. Indicate if you want to rename the selected element, all siblings with the same name or all elements with the same name.
    For example, to promote all paragraphs in your topic to sections, select Rename siblings with the same name or Rename all elements with the same name.
    Note: Siblings are elements who are located at the same level of the tree structure in your topic.
  5. Click OK.
