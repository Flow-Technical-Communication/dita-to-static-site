Inserting and deleting elements

Parent topic: Working with DITA topics

To insert an element

  1. Place your cursor where you want to insert the element.
  2. Select the correct element:
    • Press Enter and select the correct element in the displayed list of elements.
    • Select the correct element in the Elements view.
      Tip: If the Elements view is not visible in the layout of your workspace, display it by choosing Window > Show view > Elements.

To delete an element

CAUTION: When you delete an element, you also delete the content (text, table, illustration...) inside that element. If you only want to remove the element tags from a piece of content, you have to unwrap the element.
  1. Select the element that you want to delete by clicking one of the surrounding element tags.
  2. Press Backspace.
