Table components

The following additional elements can be used to build a table:

Element Description
<tgroup> Contains the column specifications, row specifications, header and body of the table.
<colspec> Contains the column specifications:
  • Column name
  • Column number
  • Cell alignment
  • Column width
<thead> Contains the header of the table.
<tbody> Contains the body of the table.
<row> Contains a single row in a table.
<entry> Contains a single cell in a table.
<sthead> Contains the header of a simple table.
Note: This element is optional.
<strow> Contains a row in a simple table.
<stentry> Contains a single cell in a simple table.
For example, the following code represents the next table: 
<table id="table_l1k_pjn_kpb">
<title>Color codes</title>
<tgroup cols="2">
<colspec colname="c1" colnum="1" colwidth="1*"/>
<colspec colname="c2" colnum="2" colwidth="3*"/>
<thead>
<row><entry>Color</entry><entry>Meaning</entry></row>
</thead>
<tbody>
<row><entry>Green</entry><entry>Keep on going</entry></row>
<row><entry>Orange</entry><entry>Stop, unless you are to close</entry></row>
<row><entry>Red</entry><entry>Stop</entry></row>
</tbody>
</tgroup>
</table>
