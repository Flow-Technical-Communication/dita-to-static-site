To assign conditions to a topic reference

This task explains how to assign values to topic references in a relationship table. If you want to assign a value to an inline link instead, you follow the same procedure as you would for any element within a topic, as described in To assign conditions to a part of a topic.
  1. Open up your DITA map in your Editor area.
  2. In the relationship table, select the <topicref> element you want to apply the conditional processing attribute to.
  3. In the Attributes list, click in the empty field next to a conditional processing attribute and enter a value.
  4. Press Enter.
    The attribute and its value are now applied to the <topicref> in the relationship table. Published topics will no longer link to the excluded topic.
