Task title

To add a title to a task topic, you use the <title> element. Titles of task topics are very important because they allow users to distinguish task topics from concept and reference topics. To make your task topics stand out:

  • Use a gerund, an imperative or a "how to" phrase in your title. Rather than using "Battery installation" as a title, for example, use "Inserting the batteries", "Insert the batteries" or "How to insert the batteries".
  • Use the same syntax for all titles. If all your task topics have a title that starts with "How to", users know which type of information to expect just by looking at the title.

For example, if you are writing a part about DITA topics and how to create them, you can use the following titles:

Title Topic
What is a DITA topic? Concept topic, including a definition of what a DITA topic is
Overview of DITA topics Reference topic, listing the different types of DITA topics that you can distinguish
Create a DITA topic Task topic, describing how you can create a new DITA topic in a particular editor (for example in Oxygen)
