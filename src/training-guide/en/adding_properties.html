Properties table

<properties>

The <properties> element gives a list of properties for the subject of the current topic, for example whether a class is public or protected. Each property contains a type, value, and description. To represent multiple values for a single type, create additional property elements and use only the <propvalue> element for each value.

The following elements can be added to the <properties> element:
  • <prophead>: headings for the properties element
  • <proptypehd>: headings for the type column of a properties table
  • <propvaluehd>: headings for the value column of a properties table
  • <propdeschd>: headings for the description column of a properties table
  • <property>: represents a single property of the current topic's subject
  • <proptype>: type of the property
  • <propvalue>: one or more values for the current property type
  • <propdesc>: short description of the property type and its listed values
