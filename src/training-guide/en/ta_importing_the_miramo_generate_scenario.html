    To import the MiramoPDF customization transformation scenario

    You need to run this procedure only if you do not have the MiramoPDF - generate new plugin transformation available in list of transformation scenarios.
    1. Select a DITAMAP in DITA Maps Manager.
    2. Choose DITA Maps > Configure transformation scenario(s)to open the list with transformation scenarios.
    3. Click the icon and choose Import scenarios.
    4. Browse for the subfolder integration\oxygen in the com.miramo.mmpdf plug-in.
    5. Import the MiramoPDF-generate-plugin.scenarios scenario.
      The MiramoPDF - generate new plugin is ready for use.
